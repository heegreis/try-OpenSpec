# try-OpenSpec

[Fission-AI/OpenSpec: Spec-driven development (SDD) for AI coding assistants.](https://github.com/Fission-AI/OpenSpec) 的 Quick Start 說明透過以下指令安裝:

```
npm install -g @fission-ai/openspec@latest
```

但本 repository 建議透過以下方式安裝明確的版本。

可先透過以下指令查詢最新版本:

```
npm view @fission-ai/openspec versions
```

再透過以下指令安裝:

```
npm install -g @fission-ai/openspec@1.3.1
```

安裝後可透過以下指令確認版本:

```
npm list -g @fission-ai/openspec
```

本 repository 的範例程式碼是基於 OpenSpec 1.3.1 版本撰寫，建議安裝此版本以確保相容性。
